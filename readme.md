# RightRSO
Right RSO is a web application built for the students of the University of Illinois at Urbana-Champaign.

## Requirements
- NodeJS
- git
- Python

## Instructions
Git clone this repo into a local directory of your choice
``` Bash
git clone https://github.com/jovanitrejo/RightRSO.git
```
Install dependencies in front end project
``` Bash
cd right_rso_frontend
npm install
```
Install dependencies in back end project
``` Bash
cd right_rso_backend
# Create virtual environment
python -m venv venv
```
IMPORTANT: Check if using windows or mac
### MacOS
```
source venv/bin/activate
```

### Windows
``` 
venv\Scripts\activate
```

Then run 
```
pip install -r requirements.txt
```
